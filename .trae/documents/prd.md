## 1. 产品概述

个人成绩差异分析工具，基于辽宁新高考（3+1+2模式）标准，帮助高中生进行历次考试成绩自我比对、目标院校差距分析与单科小分增长点挖掘，精准定位提分方向。

- 目标用户：辽宁省高中生及家长
- 核心价值：用数据驱动的方式发现成绩变化趋势、量化与目标院校的差距、找到最容易提分的知识点

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 学生 | 直接使用 | 成绩录入、分析查看、目标设定 |
| 家长 | 直接使用 | 成绩录入、分析查看、目标设定 |

### 2.2 功能模块

1. **成绩总览页**：总分趋势、各科雷达图、快速诊断
2. **成绩录入与管理页**：历次考试单科/小分录入、编辑、删除
3. **历史比对分析页**：单科/总分折线趋势、考试间差异对比
4. **目标院校分析页**：目标院校分数线设定、差距可视化、提升空间计算
5. **小分增长点分析页**：单科各题型/知识点得分率、薄弱环节定位、提分优先级排序

## 3. 核心流程

### 3.1 成绩录入流程
用户进入成绩录入页 → 选择/新建考试 → 选择选科组合（物理类/历史类） → 录入各科总分 → 录入各科小分 → 保存

### 3.2 分析查看流程
用户进入分析页 → 选择分析维度（历史比对/目标差距/小分增长） → 选择科目 → 查看可视化图表 → 获取诊断建议

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝（#1E3A5F）+ 琥珀色（#F59E0B）点缀，传达学术严谨与积极向上的氛围
- **辅助色**：浅灰背景（#F8FAFC）、白色卡片、绿色（进步#10B981）、红色（退步#EF4444）
- **按钮风格**：圆角（8px）、微阴影、hover时轻微上浮
- **字体**：标题使用思源宋体风格衬线体，正文使用无衬线体，数据数字使用等宽字体
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标**：使用lucide-react图标库，线性风格
- **图表**：使用Recharts，配色与整体风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 成绩总览页 | 总分趋势卡片 | 卡片布局，大号数字+趋势箭头+涨跌百分比，微动画 |
| 成绩总览页 | 各科雷达图 | 雷达图，六边形网格，各科得分率填充 |
| 成绩总览页 | 快速诊断 | 标签式提示，绿色进步/红色退步，带图标 |
| 成绩录入页 | 表单 | 分步表单，科目卡片式输入，小分可展开折叠 |
| 历史比对页 | 趋势图 | 折线图+面积填充，hover显示详情，支持多科叠加 |
| 历史比对页 | 差异对比 | 双向柱状图，左降右升，颜色区分 |
| 目标院校页 | 差距分析 | 水平柱状图，当前/目标双柱对比 |
| 目标院校页 | 提升空间 | 环形进度条，各科完成度 |
| 小分分析页 | 得分率热力图 | 热力图矩阵，颜色深浅表示得分率 |
| 小分分析页 | 增长点排序 | 列表卡片，按优先级排序，带进度条 |

### 4.3 响应式设计

- 桌面端优先（1440px+），适配1280px和1024px
- 平板端（768px-1024px）：侧边栏折叠为图标模式
- 移动端（<768px）：底部Tab导航，卡片单列布局
