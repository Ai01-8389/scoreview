## 1. 产品概述

个人成绩差异分析工具，基于辽宁新高考（3+1+2模式）标准，帮助高中生进行历次考试成绩自我比对、目标院校差距分析与单科小分增长点挖掘，精准定位提分方向。

- 目标用户：辽宁省高中生及家长
- 核心价值：用数据驱动的方式发现成绩变化趋势、量化与目标院校的差距、找到最容易提分的知识点
- 亮点：AI截图识别成绩录入 + 院校录取数据智能采集 + 炫丽可视化图表

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 学生 | 直接使用 | 成绩录入、分析查看、目标设定 |
| 家长 | 直接使用 | 成绩录入、分析查看、目标设定 |

### 2.2 功能模块

1. **成绩总览页**：总分趋势、各科雷达图、快速诊断、炫丽动态图表
2. **成绩录入与管理页**：截图AI识别录入 + 手动文字录入双模式、历次考试管理
3. **历史比对分析页**：单科/总分折线趋势、考试间差异对比、动态可视化
4. **目标院校分析页**：院校录取数据查询、差距可视化、提升空间计算、达标路径
5. **小分增长点分析页**：单科各题型/知识点得分率、薄弱环节定位、提分优先级排序

## 3. 核心流程

### 3.1 成绩录入流程（截图模式）
用户进入成绩录入页 → 点击截图上传 → 上传/拍照成绩单 → AI识别科目和分数 → 校验识别结果 → 手动修正（可选） → 保存

### 3.2 成绩录入流程（文字模式）
用户进入成绩录入页 → 点击文字录入 → 粘贴或输入成绩文本 → AI解析结构化数据 → 校验解析结果 → 手动修正（可选） → 保存

### 3.3 成绩录入流程（手动模式）
用户进入成绩录入页 → 选择/新建考试 → 选择选科组合 → 手动输入各科分数 → 保存

### 3.4 分析查看流程
用户进入分析页 → 选择分析维度 → 选择科目 → 查看炫丽可视化图表 → 获取诊断建议

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝（#0F172A）+ 青蓝渐变（#3B82F6→#06B6D4）+ 琥珀色（#F59E0B）点缀
- **辅助色**：深色背景（#0F172A）、半透明卡片（rgba(30,41,59,0.8)）、绿色（进步#10B981）、红色（退步#EF4444）
- **视觉风格**：深色科技风，毛玻璃效果，渐变光晕，粒子背景
- **按钮风格**：圆角（12px）、渐变背景、hover发光效果
- **字体**：标题使用粗体无衬线，数据数字使用等宽字体，大号数据展示
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局，毛玻璃卡片
- **图标**：使用lucide-react图标库
- **图表**：使用Recharts + 自定义样式，渐变填充、光晕效果、动画过渡
- **动画**：页面切换过渡、数据加载动画、hover微交互、数字滚动动画
